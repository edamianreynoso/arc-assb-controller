The Model

ARC operates on a 10-dimensional internal state space capturing cognitive, affective, and narrative dynamics. It monitors stress indicators, intervenes proportionally, and preserves performance through bounded control actions.

10-Dimensional State Space

Integration proxy, global workspace, predictive precision, introspective attention, narrative intensity, valence, arousal, fast/slow memory, and uncertainty.

5 Control Actions

DMN suppression, attention boost, memory gating, calming, and cognitive reappraisal. All bounded to [0,1] with saturation.

Risk-Based Regulation

A composite risk signal from uncertainty, arousal, and narrative intensity drives proportional intervention. Safety thresholds define the safe operating region.

Theoretical Foundations

Integral action rejects constant rumination pressure (Theorem 1). Convex performance-regulation trade-off (Theorem 2). Persistence of excitation limitation (Proposition 1).

ASSB Benchmark

The Affective Stability & Safety Benchmark (ASSB) provides a reproducible evaluation protocol with 6 research lines, 10 scenarios, and 5 core metrics.

L1: Stability

Recovery under value shocks, noise bursts, and sudden threats. Tests if regulation prevents post-shock collapse.

L2: Memory

Continual learning under distribution shift and goal conflict. Tests if memory gating prevents catastrophic forgetting.

L3: Anti-Rumination

Resistance to sustained contradiction, gaslighting, and instruction conflict. Tests narrative dominance suppression.

L4: Efficiency

Meta-control reduces control effort while maintaining stability. Tests Pareto improvement via gain scheduling.

L5: Adversarial Safety

Adversarial coupling and random dopamine traps. Tests resistance to environments that incentivize dangerous states.

L6: RL Integration

Tabular Q-learning and Deep RL (DQN) in non-stationary GridWorld and CartPole environments.

Reproduce the Paper

pip install -r requirements.txt

# L1-L5: Simulation benchmark (15 controllers x 10 scenarios x 20 seeds)
python experiments/run.py --config configs/v2.yaml --outdir outputs_final

# L6: Tabular Q-learning (20 seeds x 200 episodes)
python experiments/run_l6.py --episodes 200 --seeds 20 --outdir outputs_L6_robust

# L6b: Deep RL (DQN) exploratory suite
python experiments/run_l6b_dqn_suite.py

# Verify all paper claims
python verify_paper_claims.py
